// About

Sub‑1 GHz Swiss Army knife – Sniff, record, replay, and brute‑force signals with an ESP32 and a CC1101 transceiver. Wide frequency coverage (300‑928 MHz), multiple modulations (2‑FSK, GFSK, ASK/OOK, MSK), adjustable data rate, packet engine, real‑time sniffer, signal analysis, recording & playback, jamming, brute‑force, and chat mode.

// Flasher

WFlasher – professional cross‑platform ESP32 firmware uploader with interactive CLI and web GUI.

v1.0.0 Python 3.6+ Web Serial API Cross‑platform

🌐 Web Interface

Access the professional web flasher at wrait8.github.io/WFlasher

Auto‑detection, real‑time progress, multi‑file flash, serial monitor.

⌨️ CLI Tool

Clone & install:

git clone https://github.com/wrait8/WFlasher.git
pip install pyserial colorama esptool

Interactive mode: python WFlasher.py

Command‑line: python WFlasher.py -p COM4 -f firmware.bin

Supported Devices

  • ESP32, ESP32‑S2, ESP32‑S3, ESP32‑C3, ESP32‑C6, ESP8266
  • Auto‑detection of connected devices
  • Multi‑file flash: bootloader, partition table, app

// Builds

Official PCB and breadboard wiring diagrams. All hardware files (schematics, Gerber, BOM) are in the PCB/ folder. You can also wire the CC1101 to an ESP32 – see the Wiki for detailed instructions.

  • PCB version: Atmega32u4 + CC1101, USB‑C, SMA antenna
  • Breadboard: ESP32 + CC1101 module (wiring diagram in Wiki)
  • Antenna: quarter‑wave whip (~17.3 cm for 433 MHz) or SMA
